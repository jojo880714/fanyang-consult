# 放洋留遊學｜遊學諮詢頁面 — Claude Code Build Spec

> 把這份文件貼入 Claude Code，直接開始執行，不需要額外詢問。

---

## 一、專案目標

幫台灣留遊學代辦「放洋留遊學」建立一個**顧問面談輔助工具**。  
顧問在諮詢前由後台選好要展示的校區與卡片樣式，帶學生一起往下滑，體驗「夢想被編織」的感受，最終完成成交。

**交付物：一個 `fanyang-consult.html` 獨立檔案**
- 零外部依賴（除 Google Fonts CDN）
- 不需 server，直接在瀏覽器打開
- 目標裝置：電腦 / 平板（1280px+ viewport）

---

## 二、技術架構

```
fanyang-consult.html
├── <head>              Google Fonts（Cormorant Garamond + Noto Sans TC + DM Mono）
├── SITE_CONFIG         ← 系統注入區，只改這兩行
│   ├── style           'A' | 'B' | 'C' | 'D'
│   └── campuses        ['canary','dublin','malta']（最多 5 個）
├── CAMPUS_DATA         11 個 EP 校區完整資料
├── <style>             CSS（CSS Variables + 放洋品牌色）
├── <body>
│   ├── #advisor-bar    固定頂部工具列（顧問專用）
│   └── #main           13 個 section，往下滑
└── <script>            渲染邏輯（純 JS，無框架）
```

### URL 參數支援（方便系統直接傳值）
```
fanyang-consult.html?style=B&campuses=canary,dublin,malta
```

## 四、顧問工具列（固定頂部，#advisor-bar）

```
[放洋留遊學]  |  [卡片樣式：A B C D]  [展示校區：各校區勾選]  [卡片 ⇄ 表格]  |  [展示模式 toggle]
```

- **放洋留遊學**：Cormorant Garamond italic，深色底白字
- **卡片樣式**：A / B / C / D 四個按鈕，active 狀態用品牌色高亮。點擊即時重新渲染所有卡片
- **展示校區**：顯示系統注入的校區名稱，每個旁邊有勾選框。顧問可現場取消勾選某個校區（從頁面暫時隱藏）以聚焦討論
- **卡片 ⇄ 表格**：切換視圖，Section 02 的卡片和 Section 03 的表格實際上是同一個切換
- **展示模式**：開啟後隱藏所有 `.advisor-only` 元素（顧問備注、開發中功能等）

## 五、頁面 13 個區段詳細規格

### Section 01 — 開場 Hero（情感）

**目的**：第一眼讓學生有嚮往感，不是資訊頁。

**設計**：
- 全寬深色區塊（`--ink` 底色），高度約 380px
- 中央：`Cormorant Garamond` italic，72px，`color: rgba(255,255,255,0.95)`
- 文字依選中校區動態組合，例如：
  - 1 個校區：「你的英語旅程，從 **倫敦** 開始。」
  - 2 個：「**倫敦** 還是 **都柏林**，哪座城市在等你？」
  - 3–5 個：「選一座城市，開始說英語的人生。」並在下方用小字列出所有校區名稱
- 底部：細橫線 + 小字「往下滑，一起找到最適合你的校區」
- 右上角：品牌玫瑰色小標「EP Language School」

---

### Section 02 — 校區比較卡片（核心）

**目的**：讓學生同時看到 1–5 個校區，顧問帶著說重點。

**設計**：
- 卡片 Grid，依校區數量自動調整欄數：
  - 1 個：`max-width: 480px; margin: auto`
  - 2 個：`grid-template-columns: 1fr 1fr`
  - 3 個：`grid-template-columns: repeat(3, 1fr)`
  - 4–5 個：`grid-template-columns: repeat(auto-fit, minmax(240px, 1fr))`
- 每張卡片渲染為目前選中的樣式（A / B / C / D），見下方「六、卡片樣式規格」

---

### Section 03 — 視覺比較表格（核心）

**目的**：讓學生把所有校區欄位對齊看，理性決策。

**預設隱藏，卡片 ⇄ 表格切換時顯示（Section 02 和 03 互斥顯示）。**

**表格設計**：
- 第一欄固定（sticky left），其餘欄位可橫向捲動
- 校區名稱作為欄標題（含國旗 emoji）
- 每列有不同的視覺處理（見下表）

**表格列規格（至少 14 列）**：

| 列名 | 視覺處理 |
|------|---------|
| 所在地區 | 國旗 + 城市名文字 |
| 最低年齡 | Badge（16+ 用綠色，18+ 用藍色，30+ 用玫瑰色）|
| 校區容量 | 純文字 |
| 最短修讀 | 純文字 |
| 最大班級 | 純文字 |
| 教學語言 | 語言標籤（英文 / 德文+英文 / 法文+英文）|
| 課程週費起 | `DM Mono` 字型，加橫向費用進度條（相對最貴校區的比例）|
| 住宿週費起 | `DM Mono` 字型，加橫向費用進度條 |
| 簽證選項 | 多個小 Badge（訪客簽 / YMS / 學生簽 / 免簽90天）|
| 課程類型 | 標籤列（標準 / 密集 / 超密集 / 1-on-1）|
| 特色課程 | 玫瑰色標籤（商業認證 / 帶薪實習 / Work&Study 等）|
| 住宿選項 | 圖示 + 文字（🏠寄宿家庭 / 🏫宿舍 / 🏢公寓）|
| 費用水位 | 橫向彩色進度條（綠 → 橘 → 紅，低 → 高）|
| 適合年齡段 | 視覺年齡條（標示 16–18 / 18+ / 30+ 色塊）|

  - 一個代表性「場景句」（讓學生腦補畫面）
  - 底部：3–4 個特色 pill（icon + 文字）
- 橫向排列，多個城市並排

### Section 05 — 課程深度比較（資訊）

**目的**：讓顧問用這個區段帶學生了解各校有哪些課，怎麼選。

**設計**：
- 標題列：各校區名稱（含旗）
- 每個校區：課程卡片列表
  - 課程名稱
  - 週時數 badge
  - 週費（DM Mono）
  - 特色課程用玫瑰色邊框 + ★ 標記
- 課程可以展開（手風琴）看詳細描述
- 各校區並排（欄數同 Section 02）

---

### Section 06 — 住宿選項比較（資訊）

**目的**：讓學生了解住在哪裡，消除對「一個人住陌生地方」的不安。

**設計**：
- 各校區並排
- 每個住宿選項：類型 badge（🏠寄宿家庭 / 🏫宿舍 / 🏢公寓 / 🏨飯店）
- 週費（DM Mono）
- 距校時間
- 2 行簡短描述
- 底部：各類型的優缺點一句話總結

---

### Section 07 — 費用試算（互動）

**目的**：讓數字具體化，幫學生「算出來」而不是「猜一個」。

**設計**：
- 深色卡片（`--ink` 底色，類似 ILAC 費用卡風格）
- 所有選中校區同時試算，並排顯示
- 每個校區各自有：
  - 課程方案下拉選單
  - 住宿類型下拉選單
- 共用控制項：週數選擇（1 / 2 / 3 / 4 / 6 / 8 / 10 / 12 / 16 / 20 / 24 週）
- 即時計算：課程費 + 住宿費 + 台幣換算（含匯率說明）
- 底部：生活費參考（每月估算，依貨幣區別）

**匯率參考**：
```
£1 ≈ TWD 40
€1 ≈ TWD 35
USD 1 ≈ TWD 32
CA$1 ≈ TWD 23
AU$1 ≈ TWD 21
```

### Section 08 — 在那裡的一天（情感）

**目的**：這是「夢想編織」最關鍵的區段。讓學生閉眼想像自己的日常。

**設計**：
- 各校區並排，每個校區一條時間軸
- 時間點（`DM Mono`，玫瑰色）+ 活動標題 + 情境描述
- 時間軸以細線串連，每個節點有小圓點
- 至少 5–6 個時間點（早晨起床 → 上課 → 午餐 → 下午 → 晚餐 / 夜生活 → 週末）
- 文字語氣要「你」字當頭，讓學生代入

  - 為什麼這個校區最適合他

- 卡片底色用淡玫瑰色，邊框用品牌色

### Section 10 — 語言進步速度（資訊）

**目的**：把「能進步多少」量化，讓學生知道「X 週 = 真的有感進步」。

**設計**：
- CEFR 六級表格，每列有：
  - 等級 badge（A1 / A2 / B1 / B2 / C1 / C2，各自配色）
  - 等級名稱（初學者 / 基礎 / 中級 / 中高級 / 高級 / 精通）
  - 動態進度條（animate on scroll）
  - 所需時數估算
  - IELTS / 多益對應
- 下方互動計算器：
  - 選擇「目前程度」和「目標程度」
  - 輸出「建議修讀週數」（以密集課程 22hr/週 計算）
- 備注：每進一個 CEFR 等級約需 200 小時引導學習

**CEFR 資料**：
```

A2  基礎     bar: 18%  時數:80–180小時      IELTS:—      多益:225–549
B1  中級     bar: 35%  時數:180–400小時     IELTS:4.0–5.0  多益:550–784
B2  中高級   bar: 55%  時數:400–600小時     IELTS:5.5–6.5  多益:785–899
C1  高級     bar: 75%  時數:600–800小時     IELTS:7.0–8.0  多益:900–989
C2  精通     bar: 95%  時數:1,000+小時      IELTS:8.5–9.0  多益:990
```

### Section 11 — 你帶走什麼（情感）

**目的**：純情感收尾，讓學生想像「自己回來之後」，觸發最後的嚮往。

**設計**：
- 6 張卡片，3×2 網格
- 每張卡片：大 emoji icon + 標題 + 2–3 行說明
- 語氣要有溫度，不要功能導向

**六張卡片內容**：
1. 🗣️ **開口不再猶豫** — 沉浸式環境讓你習慣「先說再修正」，這種反應速度，在台灣補再久都練不出來
2. 🌍 **真正的國際視野** — 班上來自西班牙、日本、沙烏地、巴西……你第一次真正理解「文化差異」不只是書上的詞
3. 🧭 **對自己的認識** — 一個人在異地解決問題、適應陌生環境，你會發現自己比想像的更能扛事
4. 🤝 **跨國人脈與友誼** — 你的同學將來可能是各國的工程師、設計師、創業者，這段關係有時候是未來職涯裡意想不到的連結
5. 📄 **語言成果與紀錄** — 結業證書搭配考試成績，在履歷上是真實加分，不是「我英文還不錯」的自我感覺良好
6. 🔋 **備考的黃金視窗** — 剛回台灣的 1–2 個月，是英語狀態最好的時候。這時準備多益、雅思、托福，效率遠高於從零開始

### Section 12 — 出發前必知（資訊）

**目的**：讓學生覺得「我們幫你準備好了」，降低出發焦慮。

**設計**：
- 依照選中校區，只顯示相關資訊
- 分三子區塊：
  1. **簽證懶人包**：依各校區國家顯示對應台灣護照的簽證資訊
  2. **EP 學校認證**：列出 British Council、English UK、CRICOS 等認證 badges
  3. **出國後你不是自己一個人**：Student Experience Manager / 學生大使 / 放洋顧問後續支援

### Section 13 — 下一步 CTA（行動）

**目的**：強收尾，推進到下一步動作。

**設計**：
- 全寬玫瑰漸層橫幅（`#8C2445` → `#C8436A`）
- Cormorant Garamond italic 大字：「今天確認校區，我幫你查最近的開課日。」
